A table is a component that displays data in rows and columns.

Tables display information in a way that's easy to scan, so that users can look for patterns and insights.

<!-- ![Screenshot of a table](images/table/hero.webp "Table displaying data in rows and columns.") -->

## Usage

Use tables to display data in a structured format.

```html
<md-table>
  <md-table-header>
    <md-table-cell header>Name</md-table-cell>
    <md-table-cell header>Email</md-table-cell>
    <md-table-cell header>Phone</md-table-cell>
  </md-table-header>
  <md-table-row>
    <md-table-cell>John Doe</md-table-cell>
    <md-table-cell>john.doe@example.com</md-table-cell>
    <md-table-cell>(123) 456-7890</md-table-cell>
  </md-table-row>
  <md-table-row>
    <md-table-cell>Jane Smith</md-table-cell>
    <md-table-cell>jane.smith@example.com</md-table-cell>
    <md-table-cell>(098) 765-4321</md-table-cell>
  </md-table-row>
</md-table>
```

## Accessibility

Tables should have proper structure to be accessible to screen readers. Use the `header` attribute on table cells in the header row to indicate that they are header cells.

## Theming

Table supports [theming](../theming/README.md) and can be customized with CSS custom property tokens.
